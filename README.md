# Opportuneer

Opportuneer is a job listing website that I create using VueJS, Quasar, TailwindCSS in the frontend and PHP in the backend. It includes a custom Laravel-like router, controller classes, views, a database layer and a project structure using namespaces and PSR-4 autoloading. It highlights how to structure a PHP project without using any frameworks or libraries.
